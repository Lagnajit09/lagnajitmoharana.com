## Introduction

In today's fast-paced educational environment, students need reliable and efficient tools to enhance their study sessions. Enter the Virtual Study Buddy platform—a MERN stack application integrated with Firebase, designed to facilitate collaborative learning among students. This platform leverages cutting-edge technology to provide AI-powered study resources, real-time note-taking, community and private chat rooms, an event calendar for scheduling, and a resource-sharing hub.

## Description

Virtual Study Buddy is an innovative online platform aimed at fostering a collaborative studying environment. It integrates AI-powered study resources from renowned educational platforms, real-time note-taking capabilities, and various chat options, ensuring students have everything they need to succeed.

## Features

1. **AI-Powered Study Resources**
    - Curated, personalized study resources from platforms like YouTube, Udemy, GeeksforGeeks (GFG), Coursera, Skillshare, and TutorialsPoint.
2. **Real-Time Note-Taking**
    - Collaborative note-taking during study sessions with advanced text editing, formatting, and sharing features.
3. **Community and Private Chat**
    - Forums, chat rooms, or discussion boards for various study topics.
    - Options for both community-wide and private conversations.
4. **Event Calendar**
    - Scheduling and tracking events for users to stay organized and on top of their academic responsibilities.
5. **Resource-Sharing Hub**
    - A centralized platform for sharing educational resources, including uploading, browsing, and downloading articles, videos, tutorials, and study materials.
6. **User Management**
    - Comprehensive management of user details, ensuring a personalized and secure user experience.

### Backend

- **Node.js**: Server-side runtime environment.
- **Express.js**: Web application framework for Node.js.
- **MongoDB**: NoSQL database for data storage.
- **Mongoose**: ODM for MongoDB, handling data modeling.
- **Socket.io**: Real-time communication for features like chat and note-taking.
- **Passport.js**: Authentication middleware for Node.js.
- **Google API**: Various integrations including authentication.
- **Gemini API**: For accessing courses and their details.
- **Nodemailer**: Email sending.
- **Axios**: HTTP client.
- **Bcrypt.js**: Password hashing.
- **Cors**: Cross-Origin Resource Sharing.
- **Dotenv**: Environment variables management.
- **Express-Session**: Session management.
- **Googleapis**: Google APIs integration.
- **Jsonwebtoken**: JWT for authentication.
- **Morgan**: HTTP request logger.
- **Node-Cron**: Scheduled tasks.
- **Nodemon**: Development tool for auto-restarting server.

## Usage

- **Courses**: Navigate to the course page to browse, find, and view course details.
- **Chat**: Use the community chat for public discussions and private chat for one-on-one conversations.
- **Notes**: Create, organize, and share notes in real-time.
- **Events**: Schedule and track events using the event calendar.
- **Resources**: Upload, browse, and download educational resources from the resource-sharing hub.
